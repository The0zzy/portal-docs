title: "EnableWorldIconText"
draft: false
---
Enables or disables the text for a provided **WorldIconsItem**.  
  
_Note: There is no default text, and will need to be set before or after this property is enabled to appear_.
### Inputs
1. **WorldIconsItem**
2. **Bool**
