Returns the 1st provided value if the condition is **_True_**, otherwise, returns the 2nd provided value.
### Inputs
1. **Bool**
2. **Any Type**
    **(Output Value if the Condition is **_True_**)**
3. **Any Type**
    **(Output Value if the Condition is **_False_**)**
### Output
-   **Any Type**
