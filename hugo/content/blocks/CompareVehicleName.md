Returns a **Bool** indicating if the target **Vehicle** has the same name as the provided **VehiclesItem** or if it is the same type as the provided **VehicleTypesItem**.
### Inputs
1. **Vehicle**
2. **VehiclesItem** | **VehicleTypesItem**
### Output
-   **Bool**
